# Developer Guide

## Introduction
This document serves as a guide for developers working on the Addendas project. It contains important information regarding the development setup, coding standards, and contribution guidelines.

## Development Setup
1. **Clone the Repository**: 
   ```bash
   git clone https://github.com/limtin16/Addendas.git
   cd Addendas
   ```

2. **Dependencies**: 
   Ensure you have the following software installed:
   - Node.js
   - npm

3. **Install Packages**: 
   ```bash
   npm install
   ```

## Coding Standards
- Follow the [Airbnb JavaScript Style Guide](https://github.com/airbnb/javascript).
- Write meaningful commit messages.

## Contribution Guidelines
1. Fork the repository.
2. Create a new branch for your feature.
3. Make your changes and commit them.
4. Push to your fork and create a pull request.

## Contact
For any questions, please reach out to the project maintainers.
