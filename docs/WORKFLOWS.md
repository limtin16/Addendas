# Workflow Documentation for Main User Journeys

## User Journey: Registration
1. Navigate to the registration page.
2. Fill in the required fields (name, email, password).
3. Click on the 'Submit' button.
4. Receive a confirmation email.
5. Click on the confirmation link to activate the account.

## User Journey: Login
1. Navigate to the login page.
2. Enter registered email and password.
3. Click on the 'Login' button.
4. Access the user dashboard upon successful login.

## User Journey: Password Reset
1. Navigate to the login page.
2. Click on 'Forgot Password?'.
3. Submit the registered email for password recovery.
4. Receive an email with password reset instructions.
5. Follow the link to reset the password.

## User Journey: Order Placement
1. Browse products on the main page.
2. Click on a product to view details.
3. Add the product to the shopping cart.
4. Proceed to checkout.
5. Enter shipping and payment information.
6. Review the order and submit.
7. Receive an order confirmation email.
